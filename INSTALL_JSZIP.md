# Установка JSZip для поддержки .docx файлов

Для работы с .docx файлами необходимо установить библиотеку JSZip.

## Установка

Выполните одну из следующих команд в корневой папке проекта:

### Используя npm:
```bash
npm install jszip
```

### Используя pnpm:
```bash
pnpm add jszip
```

### Используя yarn:
```bash
yarn add jszip
```

## Проверка установки

После установки перезапустите dev сервер:

```bash
npm run dev
```

Теперь система сможет читать как .txt, так и .docx файлы!

## Альтернатива (если не хотите устанавливать JSZip)

Если вы не хотите устанавливать дополнительные зависимости, просто сохраняйте все отчёты в формате .txt перед загрузкой.
