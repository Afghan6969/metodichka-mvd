# Формат лидерского отчёта

## Описание

Лидерский отчёт - это подробный отчёт лидера фракции о работе за период времени.

## Доступ

- leader-guvd
- leader-gibdd  
- pgs-guvd
- pgs-gibdd
- gs-guvd
- gs-gibdd
- root
- super-admin

## Поля формы

1. **Ваше ФИО (лидер)** - например: "Nick_Name"
2. **Период** - дата с / дата по
3. **Разделы отчёта** - список требований, где каждое требование это раздел отчёта

## Формат отчёта

```
Отчет от лидера ГУВД Nick_Name в период с 01.10.2025 по 07.10.2025

1. Количество и доказательства проведенных собеседований на сервере
https://example.com/link1
https://example.com/link2

2. Количество принятых, уволенных ПСЖ, уволенных с ОЧС сотрудников во фракцию
Кол-во уволенных ПСЖ - 5
Кол-во уволенных с ОЧС - 2
Общее кол-во уволенных - 7
Кол-во принятых - 10

3. Количество сотрудников во фракции на момент сдачи отчета
Первые ранги - 15
Средний состав - 25
Старший состав - 8
Руководящий состав - 4, с учетом лидера
Общее количество сотрудников - 52

4. Количество проведенных обзвонов и принятых сотрудников в старший состав
Количество обзвонов за неделю - 12
Количество принятых - 3

5. Список кадровых перестановок в старшем составе
Nick_Name - повышен до Начальника ППС
Nick_Name2 - понижен с должности Зам. начальника

6. Список выданных выговоров
Nick_Name - получает выговор за нарушение 4.8 ПСГО
Nick_Name2 - получает выговор по 3.28 ПСГО

7. Фонд неустоек
Получено - 500000
Выплачено - 300000
Остаток - 200000

8. Список проведенных во фракции лекций, тренировок, RP мероприятий
Лекция по тактике - https://example.com/lecture1
Тренировка по стрельбе - https://example.com/training1

9. Мероприятия от всех филиалов организации
Мероприятие в Мирном - https://example.com/event1

10. Проведение двух межфракционных мероприятий (С непосредственным участием лидера)
Совместное патрулирование с ГИБДД - https://example.com/joint1

11. Краткая оценка работы каждого сотрудника старшего состава фракции
Nick_Name - 8/10 хорошая работа, претензий нет
Nick_Name2 - 9/10 отличная работа, претензий нет

12. Проведение генеральных построений
https://example.com/lineup1
https://example.com/lineup2
```

## Пример заполнения формы

### Поле "Название задачи" и "Ссылка":

**Раздел 1:**
- Название: `Количество и доказательства проведенных собеседований на сервере`
- Ссылка: `https://example.com/link1`

**Раздел 2:**
- Название: `Количество принятых, уволенных ПСЖ, уволенных с ОЧС сотрудников во фракцию
Кол-во уволенных ПСЖ - 5
Кол-во уволенных с ОЧС - 2
Общее кол-во уволенных - 7
Кол-во принятых - 10`
- Ссылка: `-` (если нет ссылки)

**Раздел 3:**
- Название: `Количество сотрудников во фракции на момент сдачи отчета
Первые ранги - 15
Средний состав - 25
Старший состав - 8
Руководящий состав - 4
Общее количество сотрудников - 52`
- Ссылка: `-`

И так далее для всех разделов...

## Структура разделов

1. Количество и доказательства проведенных собеседований на сервере
2. Количество принятых, уволенных ПСЖ, уволенных с ОЧС сотрудников во фракцию
3. Количество сотрудников во фракции на момент сдачи отчета
4. Количество проведенных обзвонов и принятых сотрудников в старший состав
5. Список кадровых перестановок в старшем составе
6. Список выданных выговоров
7. Фонд неустоек
8. Список проведенных во фракции лекций, тренировок, RP мероприятий
9. Мероприятия от всех филиалов организации
10. Проведение двух межфракционных мероприятий
11. Краткая оценка работы каждого сотрудника старшего состава фракции
12. Проведение генеральных построений

## Примечания

- В поле "Название задачи" можно писать многострочный текст
- Если для раздела нет ссылки, ставьте `-`
- Можно добавлять дополнительные разделы по необходимости
- Формат отчёта гибкий - адаптируйте под свои нужды
