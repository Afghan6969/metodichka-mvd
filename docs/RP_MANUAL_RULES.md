# Правила Role Play для генератора отыгровок

## Основные команды

### /me - Действия персонажа
**Назначение:** Описание действий от 3-го лица, которые нельзя показать в игре

**Правила оформления:**
- ✅ Пишется с **маленькой** буквы (перед ней имя персонажа)
- ✅ **БЕЗ** точки в конце
- ✅ Можно отыгрывать 2-4 связанных действия в одном /me

**Правильные примеры:**
```
/me достал паспорт из кармана
/me открыл багажник и достал аптечку
/me правой рукой снял рацию с поясного держателя и поднес ее ко рту
/me взял чашку со стола, после чего начал пить чай
```

**Неправильные примеры:**
```
❌ /me умер (нелогично)
❌ /me самый крутой на планете (не по формату)
❌ /me Достал паспорт. (с большой буквы и с точкой)
```

### /do - Описание окружения
**Назначение:** Описание окружающей обстановки, предметов, эмоций, состояния

**Правила оформления:**
- ✅ Пишется с **БОЛЬШОЙ** буквы
- ✅ С **точкой** в конце (или "?" для вопросов)
- ✅ Используется только когда нужно описать неочевидное
- ❌ **НЕ НУЖНО** завершать каждый /me отыгровкой /do

**Правильные примеры:**
```
/do В аптечке лежат бинты, жгут и антисептик.
/do Пульс обнаружен?
/do Что случилось с пострадавшим?
/do На листочке написано: "Встреча в 15:00".
/do Слышен стук твердого предмета о стекло.
```

**Неправильные примеры (отыгровки-паразиты):**
```
❌ /do Улыбка на лице. (и так понятно из /me)
❌ /do Рука в кармане. (и так понятно из /me)
❌ /do Передача паспорта. (и так понятно из /me)
❌ /do Аптечка в руках. (если только что отыграли взятие)
```

### /try - Действия с вероятностью
**Назначение:** Действия с определенной вероятностью успеха

**Правила:**
- ✅ Только для действий с неопределенным исходом
- ❌ **ЗАПРЕЩЕНО** для определения состояния здоровья других
- ❌ **ЗАПРЕЩЕНО** для перевода RP в свою сторону

**Правильные примеры:**
```
/try подбросил монетку, выпал орел
/try ударил бутылкой об голову так, чтобы она разбилась
/try ударив по торпеде, снова попытался завести двигатель
```

**Неправильные примеры:**
```
❌ /try посмотрев на алкотестер, увидел, что водитель пьян
❌ /try проверил пульс и увидел, что пострадавший жив
❌ /try ударив человека, привел его в чувства
```

### /b - OOC чат
**Назначение:** Внеигровые уточнения

**Примеры:**
```
/b Напишите в чат через /do, что с вами случилось
/b Можете отыграть реакцию на удар?
```

## Критические запреты

### 1. Отыгровка за другого персонажа
❌ **ЗАПРЕЩЕНО** решать за другого игрока, как его персонаж отреагирует

**Неправильно:**
```
/me нанес удар в живот Джонатану
/do Джонатан упал на землю и дергается в муках.
```

**Правильно:**
```
/me нанес удар в живот Джонатану
/do Какова реакция Джонатана?
```

### 2. Перевод RP в свою сторону
❌ **ЗАПРЕЩЕНО** выдумывать факторы в свою пользу

**Неправильно:**
```
/do Вокруг нет свидетелей.
/do У пострадавшего нет пульса.
/do В багажнике ничего нет. (когда там труп)
```

**Правильно:**
```
/do Есть ли свидетели поблизости?
/do Пульс обнаружен?
/do Что находится в багажнике?
```

### 3. Ложь в /do
❌ **ЗАПРЕЩЕНО** скрывать информацию для своей выгоды

**Пример нарушения:**
```
Игрок: /me положил деньги в кошелек
Грабитель: /do Сколько денег в кошельке?
Игрок: /do В кошельке ничего нет. ← ЛОЖЬ
```

### 4. Отыгровки-паразиты
❌ **ЗАПРЕЩЕНО** использовать бессмысленные /do после каждого /me

**Примеры паразитов:**
- `/do Улыбка на лице.` (если в /me была улыбка)
- `/do Рука в кармане.` (если в /me положили руку в карман)
- `/do Передача документа.` (если в /me передали документ)
- `/do Аптечка в руках.` (если в /me взяли аптечку)

**Когда /do НУЖЕН:**
- Описание содержимого (что в аптечке, что на листе)
- Вопросы (пульс есть? что случилось?)
- Неочевидные детали (цвет предмета, запах, звук)

## Структура медицинской отыгровки

### Правильная последовательность:

1. **Подготовка аптечки** (два варианта)
```
Вариант 1 — если есть автомобиль:
/me открыл багажник и достал аптечку
/do Аптечка в руках.

Вариант 2 — если машины нет:
/do На поясе закреплена медицинская аптечка.
/me расстегнул застёжку на поясе и взял аптечку
```

2. **Осмотр пострадавшего**
```
/me подошёл к пострадавшему
/do Что случилось с пострадавшим?
/b Напишите через /do, что с вами произошло
```

3. **Проверка состояния**
```
/me приложил руку к сонной артерии пострадавшего
/do Пульс обнаружен?
/b Ответьте: /do Да. или /do Нет.
```

4. **Медицинские действия**
```
/me достал из аптечки антисептик и бинты
/me обработал рану антисептиком
/me наложил стерильную повязку на место повреждения
```

5. **Завершение**
```
/me убедился, что состояние пострадавшего стабильное
/do Пострадавший в стабильном состоянии.
```

## Мифы (НЕПРАВДА)

❌ **МИФ:** Запрещено отыгрывать более одного /me подряд
✅ **ПРАВДА:** Можно отыгрывать хоть 10 /me подряд, если это логично

❌ **МИФ:** Каждый /me обязательно завершать /do
✅ **ПРАВДА:** /do нужен только для неочевидных деталей

❌ **МИФ:** Запрещено более двух действий в одном /me
✅ **ПРАВДА:** Можно 2-4 связанных действия

❌ **МИФ:** Запрещено использовать имена в отыгровках
✅ **ПРАВДА:** Нужно использовать имена, если много людей рядом

## Качественная отыгровка

### Хорошо:
```
/me открыл багажник и достал аптечку
/me поставил аптечку на землю и открыл её
/do В аптечке лежат бинты, жгут, антисептик и обезболивающее.
/me достал антисептик и бинты
/me подошёл к пострадавшему
/do Что случилось с пострадавшим?
/me приложил руку к сонной артерии
/do Пульс обнаружен?
/me обработал рану антисептиком
/me наложил стерильную повязку
```

### Плохо (отыгровки-паразиты):
```
/me открыл багажник
/do Багажник открыт. ← паразит
/me достал аптечку
/do Аптечка в руках. ← паразит
/me поставил на землю
/do Аптечка на земле. ← паразит
/me открыл аптечку
/do Аптечка открыта. ← паразит
/me достал бинты
/do Бинты в руках. ← паразит
```

## Реалистичность (nonRP)

❌ **ЗАПРЕЩЕНО:**
- Бегать в наручниках
- Стрелять без причины
- Игнорировать отыгровки других
- Действовать нелогично для персонажа

✅ **РАЗРЕШЕНО:**
- Использовать названия реальных городов (Москва, Санкт-Петербург)
- Упоминать реальные учебные заведения
- Ссылаться на реальных ученых, изобретателей

---

## Итог для генератора

AI должен генерировать отыгровки, которые:
1. ✅ Правильно оформлены (/me с маленькой, /do с большой)
2. ✅ Без отыгровок-паразитов
3. ✅ С вопросительными /do для уточнений
4. ✅ Без отыгровок за других персонажей
5. ✅ Логичные и реалистичные
6. ✅ С двумя вариантами (с машиной/без)
7. ✅ Короткие (8-10 команд после вариантов)
